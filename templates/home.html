Rising Waters

AI Based Flood Prediction System

Rising Waters is a Machine Learning based Flood Prediction System developed to predict whether flood conditions are likely using environmental and rainfall parameters.

The system analyses important weather and rainfall information to help users identify possible flood risks before severe conditions occur.

Start Prediction

Project Overview

Objective

Predict flood occurrence using machine learning algorithms trained on rainfall, humidity, cloud cover and seasonal weather data.

Technologies

  • Python
  • Flask
  • Machine Learning
  • Pandas
  • Scikit-Learn
  • XGBoost
  • Bootstrap 5

Dataset

The prediction model uses environmental parameters including:

  • Temperature
  • Humidity
  • Cloud Cover
  • Annual Rainfall
  • Seasonal Rainfall
  • Average June Rainfall
